# Contributing

Thank you for your interest in contributing to AI Tells.

## Current status

!!! note "Not yet accepting pull requests"

    AI Tells is in early development and is not yet accepting pull requests.

## How you can help now

While the project does not accept code contributions yet, you can still help:

<div class="grid cards" markdown>

-   :material-bug:{ .lg .middle } **Report bugs**

    ---

    If you encounter issues while using AI Tells, please open a [GitHub issue](https://github.com/tbhb/aitells/issues) with a minimal reproducible example.

-   :material-lightbulb:{ .lg .middle } **Request features**

    ---

    Have an idea that would make AI Tells more useful? Open an issue to start a discussion.

-   :material-file-document-edit:{ .lg .middle } **Improve documentation**

    ---

    Found a typo or unclear explanation? Documentation improvements are welcome.

-   :material-message-text:{ .lg .middle } **Share feedback**

    ---

    Open an issue or [discussion](https://github.com/tbhb/aitells/discussions) about how you're using AI Tells and what could be better.

</div>
