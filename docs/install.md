# Installation

AI Tells requires Python 3.13 or later.

## Basic installation

=== "pipx"

    ```bash
    pipx install aitells
    ```

=== "uv"

    ```bash
    uvx aitells
    ```

## Install from repository

To install the latest development version directly from GitHub:

=== "pipx"
    ```bash
    pipx install git+https://github.com/tbhb/aitells.git
    ```

=== "uv"

    ```bash
    uvx git+https://github.com/tbhb/aitells.git
    ```
