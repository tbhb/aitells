# Contributors

Thank you for your interest in contributing to AI Tells.

## Current status

> [!NOTE]
> AI Tells is in early development and is not yet accepting pull requests.

## How you can help now

While the project does not accept code contributions yet, you can still help:

- **Report bugs** - If you encounter issues while using AI Tells, please open a GitHub issue with a minimal reproducible example
- **Request features** - Have an idea that would make AI Tells more useful? Open an issue to start a discussion
- **Improve documentation** - Found a typo or unclear explanation? Documentation improvements are welcome
- **Share feedback** - Open an issue or discussion about how you're using AI Tells and what could be better
